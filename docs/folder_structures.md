## Tech Stack

- **Runtime**: Node.js (ES Modules вЂ” `"type": "module"`)
- **Framework**: Express 5
- **ORM**: Prisma with PostgreSQL (NeonDB)
- **Auth**: JWT (httpOnly cookie + Bearer header)
- **Password**: bcryptjs
- **Validation**: Zod
- **Dev**: nodemon, dotenv
